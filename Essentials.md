# A list of essentials commands (aka cheat sheet)

### Basic/first step

- Open a file

  ```bash
  vim file.txt
  ```

- Edit a file

  ```vim
  i      " inserts where the cursor is
  (enter some text)
  <esc>  " stops inserting

- Save & close

  ```vim
  :wq
  ```

### Get help

When lost, use

```vim
:help!
```

Or more specifically

```vim
:help :diff
```
